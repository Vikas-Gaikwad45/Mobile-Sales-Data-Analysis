# 📊 Mobile Sales Data Analysis Dashboard

## Project Overview

This project presents an interactive Power BI dashboard designed to analyze mobile sales performance across different brands, cities, payment methods, and customer ratings.

The project was developed as part of my Data Analyst training at SevenMentor, where I applied Power BI, Excel, and data visualization concepts to analyze sales data and generate business insights.

## Tools Used

- Power BI
- Microsoft Excel
- DAX

## Dashboard Features

- Brand-wise Sales Analysis
- City-wise Sales Analysis
- Customer Rating Analysis
- Payment Method Analysis
- Monthly Sales Trend Analysis
- Day-wise Sales Trend Analysis
- Interactive Filters and Slicers

## Key Insights

- Identified top-performing mobile brands.
- Analyzed customer ratings and purchasing behavior.
- Compared different payment methods used by customers.
- Evaluated sales performance across multiple cities.
- Tracked sales trends over time to support business decisions.
